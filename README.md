# Read me if you want

You can check env variables in .env.example
```env
# For sync:
DATABASE_URL=

# For auth:
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
JWT_SECRET=

# for api key encryption
API_KEY_SECRET=

# for files
UPLOADTHING_TOKEN=
```

```DATABASE_URL``` should be postgres url, I tested it with prisma postgres
